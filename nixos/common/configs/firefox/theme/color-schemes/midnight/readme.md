```
FF ULTIMA
Midnight Edition

```

To use this color scheme:
- Navigate to `about:config` page.
- Search for `user.theme`.
- Turn on `user.theme.dark.midnight`
- Make sure you only have one dark & light theme enabled at a time.
- Enjoy.
