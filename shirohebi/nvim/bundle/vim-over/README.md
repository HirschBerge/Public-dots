## Change log

* 3.0 (2015/04/01)
 * Add digraph
 * Add `<C-q>`
 * Import cmap/cnoremap setting
 * `<C-w>` by regexp
 * Multi lhs by keymapping
 * Support `<expr>` keymapping
 * Refactoring

* 2.0
 * Refactoring
 * Change cursor highlight
 * Add highlighting `:/` and `:%g/`
 * No input specital keys(e.g. `<BS>`, `<C->`)
